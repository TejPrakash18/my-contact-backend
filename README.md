## Contact-App backend (Restful APIs)

> This is Node.js and Express.js CRUD API Contact-App project, mongooDB database 

## Routes

```bash
### users

GET    /
POST   /:id

### contacts

GET      /api/contact
POST     /api/contact
GET      /api/contact/:id
PUT      /api/contact/:id
DELETE   /api/contact/:id

## Installation

```bash
# Install dependencies 
npm install

# Run in develpment
npm run dev

# Run in production
npm start
```
